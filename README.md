# RogueCrush
DASH Team2 project

"로그 크러쉬"(RogueCrush) 기획서


1. 기본 규칙 – 캔디 크러쉬(애니팡) 기반

	기본 요소는 다섯가지: 빨강,파랑,초록,보라,하양 포션

	요소들은 상단에서 계속해서 내려와 타일을 채운다.

	동일한 요소 4개 혹은 그 이상이 인접해 만나면 사라지며, 그에 따른 점수를 획득한다.

	라운드 승리 조건: 일정 목표 점수를 달성한다.


2. 로그라이크 RPG 진행 방식

	플레이어는 라운드 종료 시 캐릭터를 강화하는 특전을 획득한다.

	라운드에서 승리하지 못하면 패배로 간주되며, 보유 중인 모든 특전이 초기화된다.
	최종 승리 조건: 특정 라운드에 도달한다.

	패배하여 로비로 돌아가면, 플레이어가 최종으로 도달한 라운드 수에 비례해 가칭 	**‘코인’**을 획득한다.	

	코인은 게임 진행을 수월하게 만드는 영구적인 강화를 구매하는 데 사용한다.




3. 용어 정리

   
	피스(piece) => 매치해서 없애는 주인공들(포션)

	노드 => 하나의 피스를 가지고 관리하는 움직이는 칸

	보드(board) => 노드가 올라가는 보드 칸(움직이지 않는 칸)

	특수 블록 => 파워업: 줄 지우개, 폭탄 등 효과가 있는 블록

	장애물(obstacle) => 얼음, 돌, 젤리처럼 방해만 하는 것


	스왑(Swap) => 인접한 두 피스 교환

	매치(Match) => 같은 색의 피스 3개 연속 정렬

	롱 매치(Long Match) => 같은 색의 피스 4개 연속 정렬

	슈퍼 매치(Super Match) => 같은 색의 피스 5개 이상 연속 정렬

	콤보(Combo) => 연속 제거

	캐스케이드(Cascade) => 제거 후 낙하,리필로 추가 매치가 연쇄 발생

	스트라이프(Striped) => 가로 or 세로 한줄 전체 제거 기본적으로 롱 매치로 인해 생성됨

	랩드(Wrapped) => 기본적으로 주변 3x3 제거

	컬러밤(Color Bomb) => 스왑된 같은 색의 피스들 전부 제거
