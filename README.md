# RogueCrush
DASH Team2 project

"로그 크러쉬"(RogueCrush) 기획서


1. 기본 규칙 – 캔디 크러쉬(애니팡) 기반

	기본 요소는 네 가지: 네모, 동그라미, 삼각형, 별.

	요소들은 상단에서 계속해서 내려와 타일을 채운다.

	동일한 요소 4개가 인접해 만나면 사라지며, 그에 따른 점수를 획득한다.

	라운드 승리 조건: 일정 목표 점수를 달성한다.


2. 로그라이크 RPG 진행 방식

	플레이어는 라운드 종료 시 캐릭터를 강화하는 특전을 획득한다.

	라운드에서 승리하지 못하면 패배로 간주되며, 보유 중인 모든 특전이 초기화된다.
	최종 승리 조건: 특정 라운드에 도달한다.

	패배하여 로비로 돌아가면, 플레이어가 최종으로 도달한 라운드 수에 비례해 가칭 	**‘코인’**을 획득한다.	

	코인은 게임 진행을 수월하게 만드는 영구적인 강화를 구매하는 데 사용한다.
